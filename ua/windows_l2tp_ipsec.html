Инструкция по настройке
Zaborona VPN для windows по
протоколу L2TP+IPsec
Рекомендуется использовать систему Windows 7 или 10
1
Нажмите «Пуск» (Start) — «Панель управления» (Control panel) - «Центр управления сетями и общим доступом» (Network and sharing center):
2
Щёлкните «Настройка нового подключения или сети» (Set up a new connection or network)
3
В следующем окне выберите «Подключение к рабочему месту» (Connect to a workplace). Нажмите «Далее» (Next).
4
В следующем окне нажмите на пункт «Использовать мое подключение к Интернету (VPN)» (Use my Internet connection (VPN)). Нажмите «Далее» (Next).
5
• В поле «Адрес в Интернете» (Internet address) необходимо ввести адрес сервера для подключения: vpn.zaborona.help.
• В поле «Имя местоназначения» (Destination Name) впишите произвольное имя для этого соединения. Например, Zaborona.help PPTP
• Если за компьютером работает несколько пользователей, которым необходимо использовать данное подключение, следует отметить поле «Разрешить использовать это подключение другим пользователям» (Allow other people to use this connection)
• Отметьте поле «Не подключаться сейчас, только выполнить установку для подключения в будущем» (Don't connect now; just set it up so I can connect later)
• Нажмите «Далее» (Next).
6
В поле: «Пользователь» ("User name") введите имя пользователя: zaborona и в поле «Пароль» (Password) — пароль: zaborona. Если Вы не хотите каждый раз при подключении вводить пароль, отметьте поле «Запомнить этот пароль» ("Remember this password"). Нажмите кнопку «Создать» (Create).
7
В следующем окне нажмите кнопку «Закрыть» (Close).
8
Снова нажмите «Пуск» (Start) — «Подключение» (Connect to). Появится знакомое окошко в котором появился пункт Zaborona.help PPTP. Кликните по нему правой клавишей мыши и выберите в контекстном меню пункт Свойства ("Properties")
9
Появилось меню со свойствами подключения. Перейдите на закладку «Безопасность» (Security).
10
На закладке «Безопасность» (Security) должны быть выставлены следующие значения:
• Тип VPN: L2TP IPsec VPN;
• Шифрование данных: необязательное (подключиться даже без шифрования);
• Проверка подлинности — Разрешить следующий протокол: Протокол Microsoft CHAP версии 2 (MS-CHAP v2)
• Установите такие значения и нажмите «OK»
11
На закладке «Сеть» (Network) выбираем пункт «Протокол интернета версии 4» и нажимаем на кнопку «Свойства».
12
На закладке «Общие» (General) должны быть выставлены значения:
• Получить IP-адрес автоматически;
• Получить адрес DNS-сервера автоматически;
• Затем нажмите кнопку «Дополнительно...»
13
На закладке «Параметры IP» должны быть выставлены следующие значения:
• Стяна галочка с пункта: Использовать основной шлюз в удаленной сети;
• Поставить галочку: Отключить добавление маршрута, основанное на классе сети;
• Нажмите «OK», чтобы сохранить настройки и на следующем окне нажмите так же «OK», чтобы сохранить настройки.
14
На закладке «Сеть» выбираем пункт «Протокол интернета версии 6» и нажимаем на кнопку «Свойства».
15
На закладке «Общие» (General) должны быть выставлены значения:
• Получить IP-адрес автоматически;
• Получить адрес DNS-сервера автоматически;
• Затем нажмите кнопку «Дополнительно...»
16
На закладке «Параметры IP» должны быть выставлены следующие значения:
• Стяна галочка с пункта: Использовать основной шлюз в удаленной сети;
• Нажмите «OK», чтобы сохранить настройки и на следующем окне нажмите так же «OK», чтобы сохранить настройки.
17
Для подключения по VPN необходимо нажать «Пуск» (Start) — «Подключение к» (Connect to) и в появившемся окошке:
• дважды кликнуть левой клавишей мыши в название подключения, или
• кликнуть правой клавишей мыши в название подключения и выбрать в контекстном меню «Подключение» ("Connect")
