Інструкція з налаштування
Zaborona VPN для windows
Рекомендується використовувати Windows 7/8/10 та вище
1
Завантажте OpenVPN Community-клієнт:
WINDOWS 7/8/8.1/SERVER 2012R2 INSTALLER (NSIS)
WINDOWS 10/SERVER 2016/SERVER 2019 INSTALLER (NSIS)
Зверніть увагу, що інсталятори OpenVPN 2.4 не працюватимуть у Windows XP. Остання версія OpenVPN, яка підтримує Windows XP, - це 2.3.18, яку можна завантажити як у 32-разрядній, так і в 64-разрядній версії.
Зверніть увагу на те, що потрібен саме OpenVPN Community Клієнт. Якщо у вас є OpenVPN Connect клієнт, то він може відмовитись працювати на нестандартних портах.

Завантажте TAP-Windows Adapter V9 (Драйвер віртуальної мережевої карти):
TAP-Windows Adapter V9 INSTALLER (NSIS)
Цей драйвер потрібен для нормальної роботи програми OpenVPN. Як правило, драйвери встановлюються разом з програмним забезпеченням OpenVPN, однак в окремих випадках може знадобитись окрема установка драйвера TAP-Windows Adapter V9.
2
Завантажте файл конфігурації Zaborona VPN
Підключення до серверів Zaborona.help через Європу (для тих,у кого не працює пряме підключення):
srv0.zaborona-help_maxroutes.ovpn TCP
srv0.zaborona-help-UDP-no-encryption_maxroutes.ovpn UDP
(розширений Європа (тестовий), для доступу до Mail.ru, ОК, ВК, Касперський, Dr. Web, WebMoney та інших сайтів). Ця конфігурація розрахована більше для тих, хто не має прямого підключення до серверів Zaborona.help.
3
Запустіть OpenVPN. Натисніть правою кнопкою миші на OpenVPN у треї (значок комп'ютера внизу праворуч, біля годинника). Натисніть "Імпорт конфігурації" та виберіть завантажений файл zaborona-help.ovpn, zaborona-help_maxroutes.ovpn, srv0.zaborona-help_maxroutes.ovpn або srv0.zaborona-help-UDP-no-encryption_maxroutes.ovpn.
4
Натисніть правою кнопкою на OpenVPN значок, встановіть з'єднання.
