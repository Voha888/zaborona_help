Web-Proxy

  • Сервер, Європа:
    proxy0.zaboronahelp.pp.ua
    proxy0.zaboronahelp.ovh
    proxy0.zaborona.help Доменні імена та їх дзеркала налаштовані на декілька серверів. Рекомендовано використовувати домен замість IP-адреси, оскільки адреси серверів регулярно змінюються.
  • Порт — 3188
  • Протокол — HTTP/HTTPS
Проксі пропускає запити лише до IP-адрес перелічених у цьому списку. Тому його не вдасться використовувати для доступу до будь-яких ресурсів в Інтернеті. Переважний спосіб для комп'ютерів. Погано працює на смартфонах. Chrome:
Установіть наступне: Установки → Показати додаткові налаштування → Змінити налаштування проксі-сервера → Налаштування мережі.
Вказати параметри, які прописані вище, поля http і https.

Firefox:
Налаштуйте наступним чином: Меню → Установки → Кнопка «налаштувати…» «Проксі-сервер».
Вказати параметри, які прописані вище, поля http і https.

Opera:
Установки → Браузер → Мережа → Редагувати параметри проксі-сервера.
Вказати параметри, які прописані вище, поля http і https..

Internet Explorer:
Сервіс → Властивості браузера → Безпека → виділити «Місцева інтрамережа» → сайти → зняти галку «Всі сайти, підключення до яких виконується в обхід проксі-сервера»
Сервіс → Властивості браузера → Підключення → Налаштування мережі.
Вказати параметри, які прописані вище, поля http і https.
