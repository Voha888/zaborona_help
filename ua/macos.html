Інструкція з налаштування Zaborona VPN для MacOS
Ця інструкція підходить для MacOS 10.7.5 та новішої версії
1
Встановіть додаток Tunnelblick.
Рекомендується встановити версію Beta. Щоб інсталювати, запустіть файл з архіву. На запитання про файли конфігурації, натисніть «у мене є файл конфігурації»
2
Завантажте архів з файлом конфігурації VPN і розпакуйте його.
https://zaborona.help/zaborona-help.tblk.zip
3
Подвійним кліком по файлу налаштувань zaborona-help.tblk, встановіть профіль Zaborona VPN.
Підключіться до VPN.
Як видалити Tunnelblick з комп'ютера Mac?
Інструкція з видалення на офіційному сайті:
https://www.tunnelblick.net/cUninstall.html

  1. Завантажте додаток для видалення https://www.tunnelblick.net/release/Tunnelblick_Uninstaller_1.8_build_4405.dmg
  2. Відключіться від VPN, запустіть програму для видалення
