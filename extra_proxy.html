Web-Proxy

  • Сервер, Европа:
    proxy0.zaboronahelp.pp.ua
    proxy0.zaboronahelp.ovh
    proxy0.zaborona.help Доменные имена и их зеркала направлены на несколько серверов. Рекомендуется использовать домен вместо IP-адреса, так как адреса серверов регулярно изменяются.
  • Порт — 3188
  • Протокол — HTTP/HTTPS
Прокси пропускает запросы только к IP-адресам и доменам перечисленным в этом списке. Поэтому его не получится использовать для доступа к любым ресурсам в интернете.
Предпочтительный способ для компьютеров. Плохо работает на смартфонах.
Chrome:
Настройте следующим образом: Настройки → Показать дополнительные настройки → Изменить настройки прокси-сервера → Настройка сети.
Прописать параметры, которые указаны выше, в поля http и https.

Firefox:
Настройте следующим образом: Меню → Настройки → Кнопка «настроить…» пункта «прокси-сервер».
Прописать параметры, которые указаны выше, в поля http и https.

Opera:
Настройки → Браузер → Сеть → Изменить настройки прокси-сервера.
Прописать параметры, которые указаны выше, в поля http и https.

Internet Explorer:
Сервис → Свойства обозревателя (браузера) → Безопасность → выделить «Местная интрасеть» → сайты → снять галку «Все сайты, подключение к которым выполняется в обход прокси-сервера»
Сервис → Свойства обозревателя (браузера) → Подключения → Настройка сети.
Прописать параметры, которые указаны выше, в поля http и https.
