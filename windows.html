Инструкция по настройке
Zaborona VPN для windows
Рекомендуется использовать систему Windows 7 или 10
1
Скачайте OpenVPN Community-клиент:
WINDOWS 7/8/8.1/SERVER 2012R2 INSTALLER (NSIS)
WINDOWS 10/SERVER 2016/SERVER 2019 INSTALLER (NSIS)
Обратите внимание, что установщики OpenVPN 2.4 не будут работать в Windows XP. Последняя версия OpenVPN, которая поддерживает Windows XP, - это 2.3.18, которую можно загрузить как в 32-разрядной, так и в 64-разрядной версиях.
Обратите так же внимание на то что, нужен именно OpenVPN Community Клиент. Если у Вас есть OpenVPN Connect клиент, то он может отказаться работать на нестандартных портах.

Скачайте TAP-Windows Adapter V9 (Драйвер виртуальной сетевой карты):
TAP-Windows Adapter V9 INSTALLER (NSIS)
Данный драйвер необходим для нормальной работы программы OpenVPN. Как правило, драйверы устанавливаются вместе с программным обеспечением OpenVPN, однако в редких случаях может потребоваться отдельная установка драйвера TAP-Windows Adapter V9.
2
Скачайте файл конфигурации Zaborona VPN
zaborona-help.ovpn (только для доступа к Mail.ru, ОК, ВК, Касперский, Dr. Web)
zaborona-help_maxroutes.ovpn (расширенный, для доступа к Mail.ru, ОК, ВК, Касперский, Dr. Web, WebMoney и другим сайтам)
3
Запустите OpenVPN. Нажмите правой кнопкой на OpenVPN в трее (значок компьютера внизу справа около часов). Нажмите "Импорт конфигурации" и выберите скачанный файл zaborona-help.ovpn либо zaborona-help_maxroutes.ovpn.
4
Нажмите на значок OpenVPN правой кнопкой, установите соединение.
